## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `HF_TOKEN` environment variable in your deployment platform (Vercel) or `.env.local` for local development to your Hugging Face token
3. Run the app:
   `npm run dev`
